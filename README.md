# microgl (mgl)

Microgl is an embedded graphics library.
It aims to provide access to different displays and embedded devices.

## Support

Supported displays:
- sh1106

In progress:
- none

Supported devices:
- none

In progress:
- RPI Pico

## Getting started

Building:

```sh
make
```

```sh
make install
```
